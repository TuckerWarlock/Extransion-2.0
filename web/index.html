Playback Recording

This section will allow you to playback a previous recording for BrinkPOS software


1. Set the options as needed
2. Press the start button to begin playback
3. To finish or stop your recording, press ESC on your keyboard
4. For detailed instructions see details below

Detailed Instructions

  1. Locate the register.exe and select it so the program can launch Brink automatically
  2. Set a delay between each action (Recommended minimum of 0.5 seconds)
  3. Set how many times you want it to repeat
  4. Need to stop it early? Press ESC to stop immediately

Record Actions

This section will allow you to record actions to later repeat for BrinkPOS software


1. Click the record button below to start recording your actions.
2. This will only record mouse clicks
3. To finish or stop your recording, press ESC on your keyboard
4. For detailed instructions see details below

Start Recording mic

Detailed Instructions

  1. Locate the register.exe and select it so the program can launch Brink automatically
  2. Press the start button and wait to Brink to launch
  3. Once Brink is up and running proceed to do all the actions you want recorded
  4. Finished? Press ESC to finish the recording

Configuration

You will need to have Brink closed before you can begin recording
